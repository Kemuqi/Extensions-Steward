# Extensions-Steward

> [Changelog](changelog.md)

> Chrome Extension

## Introduction

> `Alt+Shift+Z` 可以快捷的打开扩展列表

扩展管家，快速启用、禁用扩展。禁用全部扩展和恢复被禁用的扩展
